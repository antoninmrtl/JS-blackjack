# Blackjack JS Game
Un jeu de Blackjack interactif développé en pur JavaScript. Ce projet m'a permis de mettre en pratique des concepts de programmation essentiels tout en créant une expérience utilisateur ludique et fluide.

# Fonctionnalités
Génération de Deck : Création dynamique d'un jeu de 52 cartes.

Logique de Jeu : Système de calcul des points (As, Figures, Cartes numérotées).

Interactivité : * START GAME : Initialisation de la partie.

NEW CARD : Tirage d'une nouvelle carte avec vérification automatique du score.

Gestion d'État : Affichage dynamique des messages (Win, Lose, BlackJack!) et mise à jour du DOM en temps réel.

# Stack Technique
HTML5 / CSS3 : Mise en page 'responsive' et design de table de jeu.

JavaScript (ES6+) :

Tableaux (arrays) pour le stockage du deck.

Boucles (for loops) pour la distribution.

Conditions complexes pour les règles du jeu.

Manipulation du DOM pour l'affichage des cartes.

# Comment jouer ?
Ouvrez le fichier index.html dans votre navigateur.

Cliquez sur "START GAME" pour recevoir vos deux premières cartes.

Cliquez sur "NEW CARD" pour tenter de vous rapprocher de 21 sans le dépasser.

Le jeu vous indique instantanément si vous avez gagné ou perdu !

# Concepts Validés
En développant ce projet, j'ai approfondi :

La gestion des types de données : Mapper des valeurs numériques à des noms de cartes (ex: 11 pour l'As).

Le contrôle de flux : Empêcher le joueur de tirer une carte s'il est déjà "Out" ou s'il a déjà un Blackjack.

L'organisation du code : Séparation de la logique métier (calculs) et de la logique de présentation (affichage).
